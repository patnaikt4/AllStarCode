# AI Lesson Video Feedback Tool – Web App

Next.js app for instructors and admins (auth, dashboard, upload, feedback, lesson-plan workspace).

## Running locally

1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`.
2. `npm install` then `npm run dev`.
3. Open [http://localhost:3000](http://localhost:3000).

For creating test users and assigning roles, see [../docs/setup.md](../docs/setup.md).
